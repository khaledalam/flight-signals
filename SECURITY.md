# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly by emailing khaledalam.net@gmail.com. Do **not** open a public issue.

We will acknowledge your report within 48 hours and aim to release a fix within 7 days of confirmation.

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
